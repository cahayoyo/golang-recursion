# Recursion in Golang

## Recursion Uses Stack
```
All recursion uses stack
It will call function, and returning function 1 by 1 through the stack
```

## Types Of Recursion
- Tail Recursion
- Head Recursion
- Tree Recursion
Time Complexity is O(n<sup>2</sup>)
- Indirect Recursion
- Nested Recursion
