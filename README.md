# Recursion in Golang
**Time Complexity of Recursion is _O(n)_**
## Recursion Uses Stack
```
All recursion uses stack
It will call function, and returning function 1 by 1 through the stack
```

## Types Of Recursion
- Tail Recursion
- Head Recursion
- Tree Recursion
- Indirect Recursion
- Nested Recursion
